
- The folder “scenario_1” contains the Python scripts used for the analyses of Scenario 1 in the paper. It includes four files, each corresponding to a specific number of time series in the generated collections and a specific time series length.

- The folder ''scenario_2'' contains the Python script used for the analyses of Scenario 2 in the paper.

- The folder ''scenario_3'' contains the Python script used for the analyses of Scenario 3 in the paper.

- The folder ''scenario_4'' contains the Python script used for the analyses of Scenario 4 in the paper.

- The folder ''application'' contains the Python script used for the analyses of the application in the paper. It also includes two files, ''matrix_returns.csv'', which contains the time series of log-returns used in the application (each row is a time series), and ''company_names.csv'', which gives the corresponding S&P 500 tickers.
